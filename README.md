## Setup

1. Clone the project:
```
git clone https://github.com/bwick04/webdriver-exercise.git
```
2. From the root folder, install packages:
```
npm install
```

## Running tests/scripts

- To run all tests from tasks 1, 2, 3
```
npm run wdio
```
- To run task #4:
```
npm run task4
```
- To run task #5:
```
npm run task5
```
